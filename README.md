## RU
Всем привет. Это мой локальный мессенджер, который можно развернуть, как лично на своем пк, так и на вашей локальной сети. Если у вас есть сервер, вы можете запустить файл server.py на вашем сервере с белым ip и тогда вы сможете общаться с другими пользователями через ваш сервер из любой точки мира. В планах у меня добавить шифрование E2E и возможно добавить хоть какой-то интерфейс. Пока что мессенджер не очень удобен и наверное имеет много багов. Буду благодарен если вы их найдете. Код написан на нескольких библиотеках, которые можно установить по командам снизу
```shell
pip install asyncio
pip install pynacl
```
## EN
Hello everyone. This is my local messenger, which can be deployed either on your own PC or on your local network. If you have a server, you can run the server.py file on your server with a public IP address, and then you'll be able to communicate with other users through your server from anywhere in the world. I plan to add E2E encryption and possibly some kind of interface. Right now, the messenger isn't very user-friendly and probably has a lot of bugs. I'd be grateful if you found them. The code is written using several libraries, which can be installed using the commands below.
```shell
pip install asyncio
pip install pynacl
```
