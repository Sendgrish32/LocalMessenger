<h3 align="center">
  EN 🇬🇧
</h3>

👋 Hello everyone. This is my local messenger, which can be deployed either on your own PC or on your local network. If you have a server, you can run the server.py file on your server with a public IP address, and then you'll be able to communicate with other users through your server from anywhere in the world. I plan to add E2E encryption and possibly some kind of interface. Right now, the messenger isn't very user-friendly and probably has a lot of bugs. I'd be grateful if you found them. The code is written using several libraries, which can be installed using the commands below.
```shell
pip install asyncio
pip install pynacl
```

👋 Всем привет. Это мой локальный мессенджер, который может быть развернут на вашем компьютере или в вашей локальной сети. Если у вас есть общедоступный сервер и белый IP-адрес, вы можете запустить на своем сервере server.py, и тогда вы сможете общаться с другими пользователями через свой сервер из любой точки мира. Я планирую добавить шифрование E2E и, возможно, какой-то интерфейс. Сейчас мессенджер не очень удобен для пользователя и, вероятно, имеет много ошибок, буду благодарен, если вы их найдете. Код написан на нескольких библиотеках, установить которые можно с помощью приведенных ниже команд.
```shell
pip install asyncio
pip install pynacl
```
